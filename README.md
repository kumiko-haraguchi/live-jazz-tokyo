## Getting Started
### Prerequisites

- node > 8.0.0
- yarn

### Installing

1. Clone
```
git clone https://github.com/kumiko-haraguchi/live-jazz-tokyo.git
```

2. Install
```
yarn install
```

## Running the tests

```
yarn test
```

### Coding style tests

```
yarn lint
```

## Deployment

Deployed using Heroku at [https://live-jazz-tokyo.herokuapp.com/](https://live-jazz-tokyo.herokuapp.com/)
